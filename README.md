# Viewer Boost on Termux 
> script ini untuk menambahkan jumlah Viewers pada Postingan Channel(Publik) di Telegram.

# Cara Install
> Ikuti command dibawah.

```
termux-setup-storage
```
(Skip jika sudah)
```
cd /sdcard/download/MJC-tgviews
```
```
python MJC-tgviews.py
```
